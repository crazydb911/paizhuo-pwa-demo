# 牌桌計分 PWA Demo

這是 Web/PWA demo，不需要 Apple Developer Program。

## 本機預覽

```powershell
cd C:\Users\crazydb911\Downloads\paizhuo-pwa-demo
python -m http.server 4177
```

開啟：

```text
http://127.0.0.1:4177/
```

## 給朋友測

把整個資料夾部署到任何靜態網站服務即可，例如 Cloudflare Pages、GitHub Pages、NAS Web Station，或自己的 Nginx。

正式廣告收益仍建議原生 iOS 版走 App Store；Web demo 的廣告位目前是展示殼，可再接網站廣告平台。
